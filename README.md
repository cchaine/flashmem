`flashmem` can be used to flash a binary file to an SPI flash memory using an FT232H USB to SPI chip.

## Supported flash memories

- `M25P16`

## Python dependencies

  - `pyftdi
  - `rich`

## Usage

```bash
./flashmem <binary_file>
```
